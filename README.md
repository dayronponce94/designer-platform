# Designer Platform

Plataforma profesional para gestión de proyectos de diseño con sistema de pagos integrado.

## Estructura del proyecto
- `/frontend` - Aplicación Next.js (cliente)
- `/backend` - API REST con Node.js/Express

## Stack Tecnológico
- **Frontend:** Next.js 14, React, Tailwind CSS, TanStack Query
- **Backend:** Node.js, Express, MongoDB, Mongoose
- **Autenticación:** JWT, bcrypt
- **Pagos:** Stripe
- **Almacenamiento:** Cloudinary/AWS S3
